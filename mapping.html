Mapping Agreement Provisions to Implementation Report Sections

The influence of peace agreements on implementation reports is measured by finding report sections that are semantically similar to provisions of preceding peace agreements. This process is referred to as semantic mapping where report sections are mapped by agreement provisions.

Using the interface below, select a country to see a time-ordered list of peace agreements. Selecting an agreement lists provisions that are similar to report sections. Selecting a provision lists the reports containing the mapped sections, and selecting a report lists the mapped sections in the report. Breadcrumbs provide navigation within the mapping tree.
